#2 Основи роботи з npm

  • npm init - створення нового файлу package.json з покроковими питаннями.
  • npm init -y - швидке створення package.json без запитань.
  • npm install - встановлення всіх залежностей з package.json.

  • npm install package-name - встановлення пакету локально.
  • npm install -g package-name - встановлення пакету глобально.
  • npm uninstall package-name - видалення встановленого пакету.
  • npm list - перегляд встановлених пакетів.

  • npm run script-name - запуск скрипту, визначеного в package.json.
  • npm start - запуск стандартного скрипту "start".
  • npm test - запуск тестів, якщо вони є у проєкті.

  • npm update - оновлення всіх локальних пакетів.
  • npm outdated - перевірка наявності оновлень для встановлених пакетів.
  • npm cache clean --force - очищення кешу npm.

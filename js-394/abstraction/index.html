#2 Виконати завдання

ЗАВДАННЯ 1

1) Створіть клас User із захищеними властивостями name, email, age та приватною властивістю password.

2) Додайте гетери і сетери захищених властивостей name, email, age.

3) Додайте методи для зміни пароля setPassword та перевірки пароля checkPassword.

  • 4) Створіть об'єкт класу User та перевірте роботу всіх методів.

    • ЗАВДАННЯ 2

    Створіть клас BankAccount, який буде мати приватні властивості #balance та #transactionLimit.

    Вимоги:
    1. 1) Створіть конструктор, який ініціалізує balance та transactionLimit значеннями за замовчуванням.
    2. 2) Додайте метод deposit(amount), який збільшує баланс на зазначену суму.
    3. 3) Додайте метод withdraw(amount), який знімає суму з балансу, якщо вона не перевищує transactionLimit.
    4. 4) Додайте методи getBalance та setTransactionLimit, щоб отримати баланс і змінити ліміт транзакцій відповідно.
    5. 5) Створіть об'єкт класу BankAccount та перевірте роботу всіх методів.

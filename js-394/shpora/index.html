Типи даних JS

Прості типи

  1. number - числа 1, 2, 3.5
  2. string - рядки "Якийсь текст"
  3. boolean - Логічний тип true/false
  4. null - спеціальне значення
  5. undefined - спеціальне значення
  6. Symbol
  7. BigInt

Об'єкти

  1. { } - звичайні об'єкти
  2. [ ] - Масиви
  3. String - Рядки (з методами)
  4. funtion - Функції
  5. Об'єкт дати
  6. Регулярні вирази
  7. Помилки

Функції

Function Declaration 

function nameFunction(arguments){
  //код
}
Створюється до початку виконання скрипта, можна викликати перед оголошенням.

Function Expression 

let nameFunction =  function(arguments){
 //код
};
Створюється лише тоді, коли доходить до створення змінної, можна викликати лише після оголошення.

Arrow Function 

(arguments)=>{//код}
};
Не має контексту this.

Об'єкти і масиви

Об'єкти [читати більше]

let obj = new Object();
let obj = {};
Властивості об'єктів: 
let obj = {
   name: "John",
   age: 13
};
obj.name = 'John';
Cтворення методів
(дій, функцій об'єкта) 
let obj = {
   name: "John",
   sayName: function(){
      alert(this.name);
   }
};
obj.sayName();

Масиви [читати більше]

let arr = [1,"text", [],{}];
let arr[0]=1;
Методи масивів:
  1. arr.push('a') - додає елемент в кінець масиву
  2. arr.pop() - видаляє останній елемент масиву
  3. arr.shift() - видаляє елемент з початку, зсуваючи масив
  4. arr.unshift('а') - додає елемент в початок масиву, зсуваючи масив
  5. str.split(s) - перетворює рядок в масив, s - розділювач
  6. arr.join(s) - перетворює масив у рядок, s - розділювач
  7. delete arr[2] - видаляє 3 елемент масиву
  8. arr.slice(begin, end) - копіює частину масива від begin до end індекса невключно
  9. arr.sort(fn) - сортування масиву. Якщо не передати функцію порівняння fn, то сортує як звичайні рядки по алфавіту.
  10. arr.reverse() - змінює порядок елементів на зворотній
  11. arr.concat(item1...) - створює новий масив, додаючи до масиву arr елемент item1...
Методи перебору:
  1. arr.forEach()
  2. arr.map()
  3. arr.every()
  4. arr.some()
  5. arr.filter()
  6. arr.reduce()

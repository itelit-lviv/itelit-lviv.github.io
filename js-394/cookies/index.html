#1 Теорія - рядки та cookies

Приклади функцій для роботи з кукі


function getCookie(name) {
  // Отримуємо всі кукі у вигляді одного рядка, розділяємо їх за ';'
  let cookies = document.cookie.split(';');
  
  // Перебираємо кожен кукі
  for (let cookie of cookies) {
    // Розділяємо кукі на ім'я та значення, використовуючи знак '='
    let [cookieName, cookieValue] = cookie.trim().split('=');
    
    // Перевіряємо, чи ім'я кукі відповідає запитаному (після розкодування)
    if (cookieName === decodeURIComponent(name)) {
      // Якщо ім'я співпадає, повертаємо розкодоване значення кукі
      return decodeURIComponent(cookieValue);
    }
  }
  
  // Якщо кукі з таким ім'ям не знайдено, повертаємо null
  return null;
}

function setCookie(name, value, maxAge) {
  document.cookie = `${encodeURIComponent(name)}=${encodeURIComponent(value)}; max-age=${maxAge}`;
}

#2 Виконати завдання

ЗАВДАННЯ 1

Створіть програму, яка при першому запуску буде запитувати ім'я користувача та записувати його в куки.

При наступних завантаженнях сторінки, програма повинна просто вітатися з користувачем, не запитуючи ім'я, а використовуюче ім'я, що збережене в куках.

Програма має працювати як на зображенні.

ЗАВДАННЯ 2

Продовжіть створення інтернет-магазину з попереднього уроку.

Реалізуйте можливість додавати товари в кошик та зберігайте кошик в куках сторінки.

  • Додати кнопку "Додати в кошик" для кожного товару. Додайте кнопку "Додати в кошик" в функцію getProductCard. Ця кнопка повинна мати унікальний data-id для кожного товару.
  • Створити функцію для обробки додавання товару в кошик. При натисканні на кнопку "Додати в кошик" обробити подію, отримати ID товару та зберегти його в масиві кошика.
  • Зберегти кошик у куках. Використовуйте document.cookie для зберігання даних кошика в куках.
  • При завантаженні сторінки перевірити наявність куків. Якщо кошик існує в куках, відновити його в масиві та показати користувачу.

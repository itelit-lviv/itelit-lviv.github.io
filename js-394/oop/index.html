#1 Теорія. Об'єкти

#2 Виконати завдання

  1. 1. Створіть об'єкт person з властивостями firstName, lastName, age та методом fullName, який повертає повне ім'я людини.
  2. 2. У нас є об'єкт, в якому зберігаються зарплати нашої команди: . 
    1. let salaries = {
    2.   John: 100,
    3.   Ann: 160,
    4.   Pete: 130
    5. }
     Напишіть код для підсумовування всіх зарплат і збережіть результат у змінній sum. Має вийти 390. Якщо об'єкт salaries порожній, то результат повинен бути 0.
  3. 3. У програмі створено об'єкт, який зберігає інформацію про банківську картку.
    • Виведіть інформацію про банківську карту, використовуючи метод print_card_info. Переконайтесь, що метод коректно виводить всі поля, крім методів.
    • Створіть ще один об'єкт банківської карти another_card з власними даними.
    • Створіть метод transfer. Виконайте переказ коштів з першої карти bank_card на другу карту another_card на суму 500.
    • Додайте перевірку поточного балансу перед переказом коштів. У разі нестачі коштів на балансі додайте повідомлення про це в консоль.
    4. Надіслати виконане завдання

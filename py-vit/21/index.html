#2Створення сайту на Flask та SQLAlchemy

  • pip install flask flask-sqlalchemy flask-migrate - встановлення необхідних модулів для роботи з SQLAlchemy.


from flask import Flask
from flask_sqlalchemy import SQLAlchemy
from flask_migrate import Migrate

app = Flask(__name__)

# Підключення до бази даних SQLite (можна змінити на PostgreSQL, MySQL)
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///database.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False

# Створення об'єкта бази даних
db = SQLAlchemy(app)
migrate = Migrate(app, db)

from app import db

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(80), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    posts = db.relationship('Post', backref='author', lazy=True) # Зв’язок один-до-багатьох

    def __repr__(self):
        return f'User {self.username}'

class Post(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    title = db.Column(db.String(200), nullable=False)
    content = db.Column(db.Text, nullable=False)
    user_id = db.Column(db.Integer, db.ForeignKey('user.id'), nullable=False) # Зовнішній ключ
    
    def __repr__(self):
        return f'Post {self.title}'

  • flask db init - ініціалізація міграційного сховища.
  • flask db migrate -m "Initial migration" - створення файлу міграції.
  • flask db upgrade - застосування змін до бази даних.

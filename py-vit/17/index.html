ЗАВДАННЯ ПРОЄКТУ

Нас попросили допомогти з розробкою бази даних для інтернет-магазину. Наше завдання — організувати керування базою даних та взаємодію з нею через консоль.

Виконайте наступні кроки:

Створити базу даних shop.db та таблиці Товари, Категорія, Замовлення, Кошик.

Реалізувати клас DatabaseManager для взаємодії з базою даних, який буде містити методи для:

  • Додавання даних до таблиць
  • Видалення даних з таблиць
  • Оновлення даних у таблицях
  • Отримання даних з таблиць

Реалізувати клас OnlineStore для керування інтернет-магазином через консоль, який буде містити методи для:

  • Додавання нових категорій
  • Додавання нових товарів
  • Оформлення замовлення
  • Додавання товарів до кошика
  • Перегляду товарів за категорією

Перевірити роботу всіх методів класів DatabaseManager та OnlineStore шляхом створення та взаємодії з об'єктами через консоль.

Переконатися, що можна додавати нові категорії, товари, оформлювати замовлення та переглядати товари за категорією.

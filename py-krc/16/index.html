#2 Основи Flask
Документація Flask

  • python -m venv .venv - створити нове віртуальне середовище, якщо не було створено раніше (.venv - назва середовища).
  • .venv\Scripts\activate - активація віртуального середовища, якщо немає (.venv) на початку команди в терміналі.
  • python -m pip install flask - встановлення Flask.
  • python app.py - запуск Flask-додатку.

from flask import Flask, render_template

app = Flask(__name__)  # Створюємо веб–додаток Flask

@app.route("/")  # Вказуємо url-адресу для виклику функції
def index():
    return render_template("index.html")  # html-сторінка, що повертається у браузер

if __name__ == "__main__":
    app.config['TEMPLATES_AUTO_RELOAD'] = True  # автоматичне оновлення шаблонів
    app.run(debug=True)  # Запускаємо веб-сервер з цього файлу в режимі налагодження

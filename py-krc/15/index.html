#1 Перший чат-бот

Створюємо власного чат-бота в Telegram

1) Створюємо віртуальне середовище: python -m venv venv

2) Активуємо віртуальне середовище: venv\Scripts\activate

3) Встановлення модуля python-telegram-bot: pip install python-telegram-bot --upgrade

Для створення чат-бота вам потрібно звернутися до BotFather, офіційного бота Telegram для створення і налаштування ботів. Дотримуйтесь інструкцій BotFather для створення нового бота і отримання токена доступу.

1) Використовуючи шаблон, налаштуй власного чат-бота:


from telegram import Update
from telegram.ext import ApplicationBuilder, CommandHandler

# Функція для обробки команди /start
async def start(update: Update, context):
    await update.message.reply_text("Привіт, я Telegram бот!")

# Основна частина програми
if __name__ == '__main__':
    application = ApplicationBuilder().token('YOUR_BOT_TOKEN').build()

    # Додаємо обробник для команди /start
    application.add_handler(CommandHandler('start', start))

    # Запускаємо бота
    application.run_polling()

from telegram.ext import MessageHandler, filters

# Функція для обробки будь-яких текстових повідомлень

    await update.message.reply_text(f"Ви написали: {update.message.text}")

from telegram import InlineKeyboardButton, InlineKeyboardMarkup

# Функція для відображення кнопки
async def start(update: Update, context):
    keyboard = [[InlineKeyboardButton("Натисни мене", callback_data='button_click')]]
    reply_markup = InlineKeyboardMarkup(keyboard)
    await update.message.reply_text("Натисніть на кнопку:", reply_markup=reply_markup)

# Функція для обробки натиску на кнопку
async def button_click(update: Update, context):
    query = update.callback_query
    await query.answer()  # Підтверджуємо натиск
    await query.edit_message_text(text="Ви натиснули кнопку!")

# Обробники
application.add_handler(CommandHandler('start', start))
application.add_handler(CallbackQueryHandler(button_click))

    #2 Чат-бот для реєстрації на подію

    Створити чат-бота на Python, який дозволяє користувачам зареєструватися на подію, вводячи свої контактні дані (ім'я, номер телефону, електронна пошта).

    1) Встановити Python на вашій машині.

    2) Створити віртуальне середовище:

    python -m venv venv

    3) Активувати віртуальне середовище:

    venv\Scripts\activate

    4) Встановити бібліотеку python-telegram-bot:

    pip install python-telegram-bot --upgrade

    5) Встановити SQLite для зберігання даних.

    Створити базу даних SQLite з таблицею для реєстрацій:

    CREATE TABLE registrations (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT, phone TEXT, email TEXT);

    Реалізувати основні функції бота:

    • start(): ініціює розмову і запитує ім'я користувача.
    • name(): зберігає ім'я та запитує номер телефону.
    • phone(): зберігає номер телефону та запитує електронну пошту.
    • email(): зберігає електронну пошту та записує дані в базу даних.
    • cancel(): скасовує реєстрацію.

    Запустіть бота, викликавши функцію main(), щоб він почав приймати команди від користувачів.

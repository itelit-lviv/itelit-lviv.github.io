Визначення функції

Функція (програмування) — частина програми, яка реалізує певний алгоритм і дозволяє звернення до неї з різних частин загальної (головної) програми. (Вікіпедія)

  • У функції є зазвичай ім'я.

  • Функції можна створювати і без імен.

Для чого потрібні функції?

  • Для повторного використання коду.

  • Для створення більш логічної структури програми.

  • Для об'єднання декількох дій в одну.

Корисні вбудовані функції

  • abs - модуль числа

  • min, max - мінімальне, максимальне значення

  • round - заокруглення числа

  • sum - сума елементів послідовності

  • enumerate - нумерація послідовності

Приклади:

#модуль числа

print(abs(-7)) #result: 7

#min, max

numbers = [-15, 9 ,1 ,0, 123, 11]
print(min(numbers)) #result: -15
print(max(numbers)) #result: 123

# round

print(round(10.981931248, 2)) #result: 10.99

# sum

print(sum(numbers)) #result: 129

# enumerate

winners = ['Dany', 'Alex', 'Hew']
for number, winners in enumerate(winners, 1):
print(number, winners)


#result: 1 Dany
2 Alex
3 Hew

Для чого створювати свої функції?

  • Для повторного використання коду.
  • Для створення функціоналу, якого ще нема в стандартній бібліотеці.

Як створити свою функцію?

Створюється функція за допомогою ключового слова def після чого йде назва функції name_function, вона обов'язково записується з дужками. Після цього ставиться двокрапка і в блок функції записується код. Далі для виклику функції необхідно записати її ім'я з дужками.

Приклад

def name_function():
print('Hello' * 2)

name_function() # result: HelloHello

Параметри і результат

  • Функції можна створювати без параметрів і без результату.

  • Можна також в функції сторити безліч параметрів і жодного результату.

  • Також можна створити функцію з результатом і параметрами або без параметрів.

  • Параметри слід записувати в дужках функції через кому. А при виклику функції, значення параметрів також записувати через кому.

Функція з параметрами і результатом

def name_function(age, rist):
rik = 2020 - age * rist
return rik

print(name_function(20, 175)) # result: 35000

Параметри функції

  • Параметри функції при її виклику можуть записуватись у вигляді - name_function(name, surname). Тоді параметрами функції стануть дані записані в змінні name, surname.

  • Наступний варіант запису це - name_function('Danylo', 'Ehretskyi'). Тут ми одразу передаємо дані в нашу функцію без додаткових змінних.

  • Ми також можемо поміняти порядок передачі даних. Для цього необхідно при виклику функції вказати ім'я змінної в яку ми передаємо значення: name_function(surname='Ehretskyi', name='Danylo').

  • Також, якщо при визначенні функції (не при виклику) ми задамо в нашу функцію наступне - def name_function(surname, name='Danylo'):, тоді, якщо ми не передамо в нашу функцію параметр name, він автоматично визначиться як 'Danylo'.

Змінні та область видимості

  • Змінні, створені в функції не можна викликати поза межами функції. Вони діють тільки всередині функції.

  • Наприклад:

    def name_function(name):
    full_name = name + 'Батькович'
    return full_name
    print(name_function('Zhora ')) # result: Zhora Батькович
    print(full_name)
    # result: Traceback (most recent call last):
    File "main.py", line 5, in
    print(full_name)
    NameError: name 'full_name' is not defined

  • Змінні, створені в блоці програми (не циклу, не іншої функції) можна використати в функції.

  • Наприклад:

    a = 'Mama'
    def name_function(name):
    full_name = name + 'Батькович'
    return full_name+a
    print(name_function('Zhora ')) # result: Zhora БатьковичMama

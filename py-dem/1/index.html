#2 Налаштовуємо Python у Minecraft
Завантажити Java Minecraft Python API & Spigot

Для того щоб запускати програми на Python для Minecraft потрібно:

  1. Мати встановлені наступні програми:
    • Minecraft
    • Python 3.7
    • Java 8
    • Minecraft Python API & Spigot
  2. Дозволити запуск сервера без інтернету. Для цього у папці "C:\Users\User\Documents\Minecraft Python\Minecraft Tools\server" змінюємо файл "server.properties". Потрібно задати значення параметра online-mode=false
  3. Запустити Minecraft-сервер за допомогою файлу Start_Server із папки Minecraft Tools.
  4. Запустіть Minecraft і виберіть пункт меню Мережева гра (Multiplayer). Натисніть на кнопку Додати (Add Server). У рядку введення "Назва сервера" (Server Name) назвіть ваш сервер "Minecraft Python World", а в рядку "Адреса сервера" (Server Address) введіть localhost. Потім клікніть Готово (Done). Зробіть подвійний клік по Minecraft Python World, і відкриється світ, створений сервером Spigot.
  5. Створюєте код і програмуєте свій світ Minecraft!

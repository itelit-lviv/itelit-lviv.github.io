#1 Перший чат-бот

Створюємо власного чат-бота в Telegram

1) Створюємо віртуальне середовище: python -m venv venv

2) Активуємо віртуальне середовище: venv\Scripts\activate

3) Встановлення модуля aiogram: pip install -U --pre aiogram

Для створення чат-бота вам потрібно звернутися до BotFather , офіційного бота Telegram для створення і налаштування ботів. Дотримуйтесь інструкцій BotFather для створення нового бота і отримання токена доступу.

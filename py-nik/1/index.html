#2 Розглянути теорію

Для чого потрібні змінні?

  • Змінні використовуються для зберігання даних.
  • Змінні можна повторно використовувати.
  • Можна змінювати тип і значення змінної.

Як оголосити змінну?

ім'я_змінної = значення змінної

name = 'Володимир'

Як правильно називати змінні?

Змінну необхідно називати так, щоб було зрозуміло для чого ця змінна використовується

Правильно: name, surname, age, name_person

Неправильно: a , b, ds, as, sadfads, afsd11111

Тип змінної

Тип даних визначає, що з цією змінною можна робити(які операції)
    Найпростіші типи даних:
  • Ціле число - int
  • Число з комою - float
  • Логічний тип(Правда/Брехня) - bool
  • Нічого(пустий тип) - None
  • Рядкова змінна - str

Як перевести змінну до іншого типу?

  • Рядок до цілого числа - int(word)
  • Число до рядка - str(number)
  • Інші типи аналогічно

Вивід\ввід

Вивід на екран виконується за допомогою функції print()

print('Hello world!')

Ввід даних з клавіатури виконується за допомогою функції input()

name = input('Enter your name:')
Зауваження! Тип змінної отриманої за допомогою input() - STR (а отже, якщо ви виконуєте математичні дії, змінну необхідно перевести до типу числа)

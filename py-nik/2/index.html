#2 Розглянути теорію

Для чого потрібні умовні оператори?

  • Початок казки
  • Наліво підеш - коня втратиш
  • Направо підеш - життя втратиш
  • Кінець казки

Оператор if

  • Початок казки
  • if Наліво підеш: коня втратиш
  • Кінець казки

Блоки і відступи в Python

  • Блок коду - логічно згрупований шматок коду
  • В python нема - {}, які розділяють блоки коду
  • Тому для кожного блоку коді потрібно робити ВІДСТУП
  • Відступ - 4 пробіла(кнопка TAB зліва на клавіатурі)

if - elif - else

  • if перша умова:
    • Код 1
  • elif друга умова:
    • Код 2
  • else:
    • Код 3
    Якщо не виконується перша умова, то перевіряється друга умова, якщо обидві БРЕХНЯ, то виконується блок else.

Внутрішній if

  • Всередині if-elif-else може бути інший if.
  • Таких вкладень може бути скільки завгодно

Функція strip()

Функція strip() створена для видалення пробілів зліва і справа.

Також є функції lstrip() та rstrip() створені для видалення пробілів зліва і справа відповідно.

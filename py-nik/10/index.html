ЗАВДАННЯ 1

Написати програму, яка відкриває текстовий файл для читання та запису, а потім:

  • Спробує прочитати вміст файлу.
  • Якщо файл не існує, програма створить новий файл з певним початковим текстом.
  • Якщо відбудеться помилка під час зчитування файлу, програма виведе відповідне повідомлення.
  • Після чого програма спробує записати у файл новий текст.

ЗАВДАННЯ 2

Створення та запис даних у файл

Створіть програму, яка запитує у користувача ім'я та прізвище. За допомогою блоку try/except запишіть цю інформацію у текстовий файл `users.txt`. Перехопіть й обробіть можливі помилки, такі як неможливість відкрити файл або неправильний формат введених даних.

Підказки:

  • Використовуйте функцію `open()` для відкриття файлу в режимі `w` (запис).
  • Запишіть дані користувача у файл за допомогою методу `write()`.
  • Оберніть код запису в блок try/except.
  • У блоці except обробіть можливі помилки та виведіть відповідні повідомлення користувачу.

ЗАВДАННЯ 3

Читання та обробка даних з файлу

Створіть програму, яка читає дані з текстового файлу `users.txt`. Виведіть на екран ім'я та прізвище кожного користувача з нового рядка. Обробіть можливі помилки, такі як неможливість відкрити файл або відсутність даних у файлі.

Підказка:

Прочитайте дані з файлу за допомогою методу readlines().

Додаткові завдання:

  • Додайте до програми можливість оновлення даних користувача.
  • Реалізуйте функцію пошуку користувача за ім'ям або прізвищем.
  • Додайте можливість видалення користувача з файлу.

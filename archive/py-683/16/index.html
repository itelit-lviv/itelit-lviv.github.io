Тема 16. Свій блог на Flask. Робота з базами даних за допомогою SQLAlchemy. СУБД SQLite.

Початок роботи з мікрофреймворком Flask


  • python -m venv venv або python3 -m venv venv- створити віртуальне середовище.

  • venv\Scripts\activate - активувати віртуальне середовище.

  • pip install Flask або pip3 install Flask - встановити модуль Flask.

  • pip install flask-login або pip3 install flask-login - встановити модуль FlaskLogin.

  • pip install pip install flask_sqlalchemy - встановити модуль Flask-SQLAlchemy для роботи з базами даних.

Що ми вивчили з Flask'у?


Основні команди для роботи з Flask'ом

  • app = Flask(__name__) - створили екземпляр головного класу.
  • @app.route() - за допомогою функції route() декоратора нашого екземпляру програми app відслідковуємо URL-адресу.
  • app.run(debug=True) - запуск веб програми в режимі виведення помилок.

Шаблонізатор Jinja2

Команди Jinja2:

  • {% block name %}
    {% endblock %}     - створення блоку в базовому шаблоні, яке буде розширювати (доповнювати) інший файл.

  • {% extends 'file.html' %}
    {% block name %}
    {% endblock %}     - підключення базового шаблону, та вставлення блоку в нього.

Настройка Flask-SQLAlchemy

Для початку підключаємо модуль Flask-SQLAlchemy

from flask_sqlalchemy import SQLAlchemy
from flask import Flask

Потім створюємо веб-додаток та вказуємо підключення до бази даних:

app = Flask(__name__)

app.config['SECRET_KEY']='b9cf9cc366f1311cb1b9ddc540f393eaaab79ba5'
app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///app.db'
app.config['SQLALCHEMY_TRACK_MODIFICATIONS']=False

db = SQLAlchemy(app)

Робота з базою даних за допомогою SQLAlchemy

  • res = Users.query.all() - вибрати всі записи з таблиці 'users'
  • f = Users.query.first() - вибрати перший запис з таблиці 'users'
    • Вибрати дані за фільтром:
  • Users.query.filter_by(id = 1).all()
  • Users.query.filter(Users.id == 1).all()
  • Users.query.filter(Users.id > 1).all()
  • Users.query.limit(2).all() - вибрати максимум 2 результати з таблиці 'users'
  • Users.query.order_by(Users.email).all() або Users.query.order_by(Users.email.desc()).all() - вибрати відсортовані за певним полем рядки таблиці
  • Users.query.get(2) - отримати користувача за допомогою ID

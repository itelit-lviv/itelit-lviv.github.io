Робота з файлами

В Пайтон можна відкривати будь-які файли та працювати з ними.

Відкриття файла в Python

За допомогою функції open можна відкривати файл в середі Пайтон і виводити його вміст на екран.

Для відкриття файла test.txt напишіть наступне:

test_file = open('test.txt')
text = test_file.read()
print(text)

Запис в файл

Функція open відкриває файл тільки для читання, тому для запису інформації в файл необідно писати уточнення 'w', наступним чином:

text_file = open('test.txt','w')

Якщо такого файлу не існує, то він створиться. Далі записати інформацію в файл можна за допомогою функції:

text_file = open('test.txt','w')
text_file.write('Some text')

Після чого файл обов'язково треба файл закрити:

text_file = open('test.txt','w')
text_file.write('Some text')
test_file.close()

Функції модуля random

  • random.randint(A, B) - випадкове ціле число N, A ≤ N ≤ B.
  • random.choice(list) - випадковий елемент непустого масиву (список, кортеж, бібліотека).
  • random.random() - випадкове число від 0 до 1.
  • random.uniform(A, B) - випадкове неціле число float, A ≤ N ≤ B (або B ≤ N ≤ A).

Модуль time

Функція time() повертає кількість секунд, які пройшли від півночі 1 січня 1970 року. Це можна використовувати для визначення часу роботи програми. Засікаємо час до виконання і після. Далі виводимо різницю цих значень і отримуємо час роботи програми.

import time
t1 = time.time()
for i in range(0,1000):
print(i)
t2 = time.time()
print(t2-t1)

Отримуємо дату й час за допомогою localtime()

Функція localtime() повертає дату й час у вигляді кортежу.Для того, щоб надрукувати час або рік. Необхідно записувати наступним чином:

import time
t = time.localtime()
print(t[0])
print(t[3])

Кортеж формується наступним чином - (рік, місяць, день, година, хвилина, секунда, який день тижня[неділя - 0, далі 1,2...6], який день року від 1 до 366.)

Функція sleep()

Функція sleep() потрібна для того, щоб заповільнити час виконання програми. Наприклад для виведення чисел від 1 до 60 можна використати такий цикл:

import time
for i in range(1,61):
print(i)
time.sleep(1)

Таким чином ми зробили секундомір, який рахує від 1 до 60.

Третій метод форматування рядків

Якщо вам необхідно вписати у ваш рядок інші змінні числові або рядкові, то можна використовувати цей метод:

a = 3
b = 'apple'
c = 'I find %s %s'%(a,b)
print(c)
#result: I find 3 apple

Для виведення інформації краще використовувати метод через f, якщо необхідно створити новий рядок, який включає інші змінні, то краще цей.

Робота з файлами

В Пайтон можна відкривати будь-які файли та працювати з ними.

Відкриття файла в Python

За допомогою функції open можна відкривати файл в середі Пайтон і виводити його вміст на екран.

Для відкриття файла test.txt напишіть наступне:

test_file = open('test.txt')
text = test_file.read()
print(text)

Запис в файл

Функція open відкриває файл тільки для читання, тому для запису інформації в файл необідно писати уточнення 'w', наступним чином:

text_file = open('test.txt','w')

Якщо такого файлу не існує, то він створиться. Далі записати інформацію в файл можна за допомогою функції:

text_file = open('test.txt','w')
text_file.write('Some text')

Після чого файл обов'язково треба файл закрити:

text_file = open('test.txt','w')
text_file.write('Some text')
test_file.close()

Модуль Copy та його використання

Уявимо, що у нас є клас Animals, з ініціалізацією:

class Animals:
def __init__(self, vid, lapy, color):
self.vid_an = vid
self.lapy_an = lapy
self.color_an = color

Тепер ми можемо створити об'єкт цього класу. Нехай це буде Стів.

steve = Animals('pauk', 6, 'silver')

Тепер для того, щоб створити зграю пауків, краще буде не створювати нові об'єкти, а скопіювати цей:

import copy
class Animals:
def __init__(self, vid, lapy, color):
self.vid_an = vid
self.lapy_an = lapy
self.color_an = color
steve = Animals('pauk', 6, 'silver')
albert = copy.copy(steve)
anton = copy.copy(steve)

Але якщо ми захоемо скопіювати список з нашими об'єктами, то при зміні першого списку, буде змінюватись і другий. Для того, щоб цьому запобігти, необхідно використати функцію deepcopy. При використанні цієї функції при зміні першого списку, другий буде незмінним.

Модуль sys

Модуль sys містить засоби для керування та взаємодії з самим інтерпертатором Python.

Об'єкт stdin

У об'єкта stdin є функція readline(), за допомогою якої можна брати інформацію від користувача. Вона прцює так само, як і input, головна відмінність їх в тому, що в readline() можна вказувати кількість символів, яку вона отримає:

import sys
print('Enter your str:')
v = sys.stdin.readline(15)
print(v)
#result:
Enter your str:
Hello, my name is Danylo Romanovych!
Hello, my name

Об'єкт stdout

На відміну від stdin, stdout призначений для виводу інформації. Він дозволяє робити те, що й print, але stdout - файловий об'єкт, у якого є функції, наприклад write():

import sys
sys.stdout.write('Hi, my name is Danylo.')
#result:
Hi, my name is Danylo.23

Функція write() повертає кількість символів, які вивели. Саме тому в кінці є цифра 23. Це можна використовувати для того, щоб рахувати скільки взагалі символів ми вже вивели користувачу.

Модуль time

Функція time() повертає кількість секунд, які пройшли від півночі 1 січня 1970 року. Це можна використовувати для визначення часу роботи програми. Засікаємо час до виконання і після. Далі виводимо різницю цих значень і отримуємо час роботи програми.

import time
t1 = time.time()
for i in range(0,1000):
print(i)
t2 = time.time()
print(t2-t1)

Отримуємо дату й час за допомогою localtime()

Функція localtime() повертає дату й час у вигляді кортежу.Для того, щоб надрукувати час або рік. Необхідно записувати наступним чином:

import time
t = time.localtime()
print(t[0])
print(t[3])

Кортеж формується наступним чином - (рік, місяць, день, година, хвилина, секунда, який день тижня[неділя - 0, далі 1,2...6], який день року від 1 до 366.)

Функція sleep()

Функція sleep() потрібна для того, щоб заповільнити час виконання програми. Наприклад для виведення чисел від 1 до 60 можна використати такий цикл:

import time
for i in range(1,61):
print(i)
time.sleep(1)

Таким чином ми зробили секундомір, який рахує від 1 до 60.

Модуль pickle

Модуль pickle використовується для запису даних в файли в більш зручний формат для запису і чистання. Наприклад pickle знадобиться, якщо розробляєте гру і хочете записувати її дані на диск. Нехай у вас є словник з даними про персонажа, його місце на карті, кількість грошей і тд. Тепер цей словник можна зберегти в файлі за допомогою функції dump().

import pickle
game_data = {'side': 'A50,B123',
'pockets':['keys','knife','rock'],
'backpack':['humer', 'apple', 'juice'],
'money': 1231.123 }
save_file = open('data.dat', 'wb')
pickle.dump(game_data, save_file)
save_file.close()

В 6 рядку в функції open() аргумент 'wb' означає, що ми відкриваємо фал для запису в двійковому(бінарному) форматі.

Об'єкти створені за допомогою pickle можна знову завантажити і прочитати за допомогою функції load():

import pickle
load_file = open('data.dat', 'rb')
text = pickle.load(load_file)
load_file.close()

Третій метод форматування рядків

Якщо вам необхідно вписати у ваш рядок інші змінні числові або рядкові, то можна використовувати цей метод:

a = 3
b = 'apple'
c = 'I find %s %s'%(a,b)
print(c)
#result: I find 3 apple

Для виведення інформації краще використовувати метод через f, якщо необхідно створити новий рядок, який включає інші змінні, то краще цей.

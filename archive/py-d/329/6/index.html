Що таке списки?

Грубо кажучи, списки використовують для зберігання багатьох рядків. Це масив.

Як будуються списки?

Списки будуються дуже просто:

name_list = ['first' , 2 , 'third' , 'hello', 12.33, True]

Особливості списків

До елементів списку можна звертатись за індексом.

name_list[2] - Виведе нам на екран 3 елемент нашого списку

Також елементи списку можна змінювати. Для цього необхідно записати таке:

name_list[2] = 'Danylo'

 До списків також можна використовувати зрізи!

В середині списку можна створити інші списки. Такі списки вважаються двовимірними.

Арифметика списків

  • Ми можемо додавати між собою списки, після чого вони стануть одним списком:

    a = ['abc', 2]
    b = [1,2]
    print(a+b)
    Result: ['abc', 2, 1, 2]

  • Ми можемо множити список на ціле число, результатом буде список з елементами, які повторюються декілька разів:

    a = ['abc', 2]
    print(a*3)
    Result: ['abc', 2,'abc', 2,'abc', 2]

Функції для списків

  • list.append(x) - Добавляє елемент в кінець списку.
  • del list[i] - Видаляє і-тий елемент зі списку.
  • list.extend(L) - Розширює список list, добавляючи в кінець списку всі значення списку L.
  • list.insert(i, x) - Вставляє на і-тий індекс елемент х.
  • list.remove(x) - Видаляє перший елемент списку зі значенням x. ValueError, якщо такого елементу не існує.
  • list.pop([i]) - Видаляє i-тий елемент і повертає його. Якщо індекс не вказаний, видаляється останній елемент.
  • list.index(x, [start [, end]]) - Повертає індекс першого елемента зі значенням x (при цьому пошук ведеться від start до end).
  • list.count(x) - Повертає кількість елементів зі значенням x.
  • list.sort([key=функція]) - Сортує список на основі функції.
  • list.reverse() - Перевертає список
  • list.copy() - Поверхнева копія списку
  • list.clear() - Очищує список

Що таке кортеж?

Кортеж дуже схожий на списки, за винятком двох особливостей.
  1. Записується він в круглих дужках:

    cortezh = ('a', 'hello', 1 ,2 ,34,5)

  2. Кортеж не можна змінювати! Тобто замінити, додати чи видалити елемент як в списках не можна!

Що таке словник?

Словник це щось дуже схоже на списки. Тільки знову є деякі відмінность.

Словник

  • Будується наступним чином:

    library_name = {'Перший ключ': 'Перше значення',
    'Другий ключ': 'Друге значення'}

  • Кожному значенню відповідає в бібліотеці не індекс, а ключ.

  • Звертатись до елементів в бібліотеці необхідно за ключем:

    print(library_name['Перший ключ'])

  • Видаляти елементи в бібліотеці також необхідно за ключем:

    del library_name['Перший ключ']

  • Об'єднювати бібліотеки не можна. Видасть помилку.

Модуль random

Модуль random надає функції для генерації випадкових чисел, букв, випадкового вибору елементів послідовності.

Що таке словник?

Словник це щось дуже схоже на списки. Тільки знову є деякі відмінності.

Функції модуля random

  • random.randint(A, B) - випадкове ціле число N, A ≤ N ≤ B.
  • random.choice(list) - випадковий елемент непустого масиву(список, кортеж, бібліотека).
  • random.random() - випадкове число від 0 до 1.
  • random.uniform(A, B) - випадкове неціле число float, A ≤ N ≤ B (або B ≤ N ≤ A).

Що ми зробили?


На першому занятті ми ознайомились з теорією про Бази даних. Вихначили, які вони бувають та де які потрібно використовувати. Почали вивчати SQL

SQL запити


  • CREATE TABLE name - створити таблицю name

  • DROP TABLE name - видалити таблицю name

  • INSERT INTO name VALUES(*values*) - вставити в таблицю name значення values

  • UPDATE name SET colname = value WHERE colname = value - оновити в таблиці name значення в colname - value, де colname = value

  • SELECT * FROM name - вибрати все з таблиці name

  • WHERE colname = value ORDER BY ASC/DESC - де значення в colname = value сортувати за зростанням/спаданням

  • WHERE colname BETWEEN value AND value - де значення в colname знаходяться в діапазоні між value i value

  • WHERE colname IN (*values*)де значення в colname знаходяться в списку values

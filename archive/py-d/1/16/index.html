Які команди ми вивчили по MongoDB?


  • limit(count) - дозволяє зробити вибірку з обмеженою кількістю елементів.
  • db.collection.find({},{_id:0}) - за допомогою другого аргументу функції find, можна обмежити кількість полей, які будуть виведені.
  • sort({age:1/-1}) - сортувати дані по певному полю, 1 - по зростанню, -1 - по спаданню.
  • sort({age:1/-1,email:1}) - можна вказувати безліч елементів.
  • db.collection.find({age:22},{_id:0}) - перший аргумент функції find - це фільтр для вибірки даних.
  • find({$or:[{age:22},{email:’fow@wwf.oe’}]}) - функці або в функції find.
  • db.collection.find({$or:[{age:{$lt/gt:28}},{email:’fow@wwf.oe’}]},{_id:0}) lt gt lte gte eq ne - якщо необхідно зробити вибірку, де, наприклад, вік має бути більшим чи меншим за певне число використовуються функції $some, де на місці some може бути: lt - менше, gt - більше, lte - менше рівно, gte - більше рівно, eq - рівно, ne - не рівно.
  • db.collection.find({email:{$in:[’fow@wwf.oe’]}}) nin - функція in шукає значення, які містяться в даному списку, її проилежність функція nin.
  • db.collection.find({child:{$exists:true}},{_id:0}).limit(count) - функція exists робить вибірку елементів, в яких таке поле існує,або елементів, в яких такого поля нема.
  • db.collection.find({child:{$size:2}},{_id:0}).limit(count) - функція size робить вибірку елементів у яких в одному полі масив з певною кількістю елементів.
  • find({“child.1”:”ddewd”}) - за допомогою цього виразу ми робимо вибірку даних, де в елемента в полі child другий елемент масиву дорівнює ddewd.
  • updateOne({filter},{$set:{name:’name’}}) - функція для оновлення одного документу, може бути з закінченням Many, для багатьох документів.
  • replaceOne({age:23},{data}) - функція для заміни одного документа іншим документом, може бути з закінченням Many, для багатьох документів.
  • deleteOne({filter}) - функція для видалення одного документу, може бути з закінченням Many, для багатьох документів.
  • createIndex({title:’text’}) - створення індексу для розширення можливостей.
  • db.collection.find({$text: {$search: ‘text’}}) - після створення індексу, можна шукати певне слово в тексті в одному зполей документу.
  • db.collection.find({$text: {$search: ‘text’},score:{$meta:’textScore’}}) - тут ми додаємо параметр score, який визначить наскільки наш документ підпадає пошуку.
  • db.collection.count({filter}) - функція для виведення кількості документів у вибірці.
  • db.collection.distinct(’data’) - виводить унікальні значення полей з вибірки.

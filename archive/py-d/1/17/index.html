Початок роботи з мікрофреймворком Flask


  • pip install flask - команда для встановлення мікрофреймворку.
  • pip install flask_sqlalchemy - допоміжна бібліотека для роботи з базами даних.

Що ми вивчили з Flask'у?


Основні команди для роботи з Flask'ом
  • app = Flask(__name__) - створили екземпляр головного класу.
  • @app.route() - за допомогою функції route() декоратору нашого екземпляру відслідковуємо URL.
  • app.rub(debug=True) - запуск веб програми в режимі виведення помилок.

Шаблонізатор Jinja


За допомогою шаблонізатору можна скоротити кількість файлів і під'єднати до готової головної сторінки.

Тож можливості Jinja:

  • {% block name %}{% endblock %} - створення місця, куди буде підключатись інший файл.
  • {% extends 'file.html' %}
    {% block name %}{% endblock %}     - підключення головного файлу, та вставлення блоку в нього.

Робота з SQLite

Визначаємо файл бази даних: 

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///cinema.db'
			app.config['SQLALCHEMY_TRACK_MODIFICATIONS'] = False
			db = SQLAlchemy(app)

Далі за допомогою класу визначаємо таблицю, яка буде в нашій базі даних:

class Users(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    email = db.Column(db.String(30), nullable=False)
    login = db.Column(db.String(20), nullable=False)
    password = db.Column(db.String(20), nullable=False)
    name = db.Column(db.String(30), nullable=False)
    surname = db.Column(db.String(30), nullable=False)
 
    def __repr__(self):
        return '<Users %r>' % self.id

Далі для того щоб передати дані в цю таблицю, необхідно створити форму, та визначити метод POST для передачі даних і вставити дані в таблицю:

if request.method == 'POST':
        email = request.form['inputEmail']
        login = request.form['inputLogin']
        password = request.form['inputPassword']
        name = request.form['inputName']
        surname = request.form['inputSurname']
 
        user = Users(email=email,login=login,password=password, name=name, surname=surname)
 
        try:
            db.session.add(user)
            db.session.commit()
            return redirect('/')
 
        except:
            return 'Помилка при реєстрації'

Модуль tkinter

Малювання за допомогою черепашки має один головний недолік - черепашки рухаються дуже повільно.

Створюємо кнопку

Давайте для початку використаємо tkinter та створимо програму з кнопкою. Для цього уведіть наступний код:

from tkinter import *
t = Tk()
btn = Button(t, text='press')
btn.pack()
tk.mainloop()

В першому рядку ми імпортували модуль tkinter. Конструкція from модуль import * дозволяє звертатись до вмісту модуля, не вказуючи його ім'я щоразу.

В другому рядку ми створили змінну і зберегли в ній об'єкт класу Тк. Об'єкт t створює пусте вікно, в якому можна добавляти кнопки, рядки вводу і т.п.

В третьому рядку ми створили кнопку викликом btn = Button, де першим аргументом передали змінну t, а другим рядок 'press'. Для того, щоб кнопка з'явилась на екрані необхідно написати btn.pack(), яка включає відображення кнопки.

Від кнопки 'press' не багато користі. Давайте це змінимо. Створимо функцію, яка в наш командний рядок буде виводити "Привіт" і використаємо її з кнопкою:

from tkinter import *
def hello():
print("Привіт")
t = Tk()
btn = Button(t, text='press', command=hello)
btn.pack()
tk.mainloop()

Створюємо полотно для малювання

Для створення малюнків на екрані, від кнопок сенсу небагато, для цього використовують полотно - об'єкт класу Canvas, який так само входить в наш модуль. Для створення полотна необхідно задати ширину і висоту:

from tkinter import *
t = Tk()
canvas = Canvas(t, width=400,height=400)
canvas.pack()
t.mainloop()

Малюємо лінії

В tkinter для малювання ліній на полотні використовуються координати в пікселях. Оскільки наше полотно займає 400 пікселів в ширину і 400 пікселів в висоту, його праий нижній кут має координати(400,400). Отже лінію можна наалювати за допомогою функції create_line:

from tkinter import *
t = Tk()
canvas = Canvas(t, width=500, height=500)
canvas.pack()
canvas.create_line(0,0,400,400)
t.mainloop()

Малюємо прямокутники

За допомогою tkinter малювати прямокутники набагто простіше ніж за допомогою черепашки. Всьо, що необхідно знати - це координати кутів прямокутника.

from tkinter import *
t = Tk()
canvas = Canvas(t, width=500, height=500)
canvas.pack()
canvas.create_rectangle(10,10,40,40)
t.mainloop()

Координати, які ми передали цій функції - це координати лівого верхнього кута і правого нижнього.

Кольорове малювання

Ми хочем використовувати різні кольори в наших малюнках. Для цього використовується атрибут fill:

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_rectangle(10,10,40,40,fill='red')
t.mainloop()

Малювання дуг

Для того щоб намалювати дугу, необхідно викликати функцію create_arc, передавши їй координати кутів прямокутника, в який вона буде вписана:

canvas.create_arc(10,10,200,100,extent=180, style=ARC)

Лівий верхній кут описаного прямокутника буде розташований в точці з координатами (10,10), а правий нижній в точці (200,100). Далі вказаний аргумент extent, в якому передається кут поворота дуги в градусах.

Малювання багатокутників

Для того, щоб створити багатокутник використовується функція create_polygon, де аргументами виступають точки багатокутника:

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_polygon(10,10,100,40,100,140)
t.mainloop()

Відображення тексту

Для того, щоб вивести текст на полотні, використовується функція create_text, яка приймає дві координати х- та у-позиції напису, а також аргумент з самим текстом.

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_text(10,10,text='Hello',fill='red')
t.mainloop()

В аргументі font можна задати кортеж, в якому будуть назва і розмір шрифту. Розмір шрифту вказується в пунктах font=('Colibri', 20), для того щоб вказати в пікселях, необхідно задавати від'ємне значення - ('Colibri', -20).

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_text(10,10,text='Hello',fill='red',font=('Colibri', -20))
t.mainloop()

Зображення

Для виведення зображень використовують функцію create_image (Використовується тільки для .gif. Для того, щоб вставити PNG або JPG формати, необхідно використовувати іншу бібліотеку або модуль):

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
my_image = PhotoImage(file='test.gif')
canvas.create_image(0,0,anchor=NW,image=my_image)
t.mainloop()

Аргумент anchor=NW означає, що зображення має виводитись з верхнього лівого кута (інакше воно виведеться з центру зображення). Останній аргумент задає саме зображення, тобто вказує на фото.

Створення простої анімації

Модуль не розрахований на створення анімації, але щось просте ми зробити можемо.

import time
from tkinter import *
tk = Tk()
canvas = Canvas(tk, width=400, height=200)
canvas.pack()
canvas.create_polygon(10, 10, 10, 60, 50, 35)
for x in range(0, 60):
canvas.move(1, 5, 0)
tk.update()
time.sleep(0.05)
tk.mainloop()

Як працює анімація? Задаємо звичайний цикл, який рахує від 0 до 59. Код, який знаходить в блоці цього циклу відповідає за переміщення трикутника. Функція canvas.move(1, 5, 0) рухає об'єкт, ідентифікатор якого дорівнює 1, на 5 пікселів вправо і на 0 пікселів вниз. Функція tk.update() використовується для змушеного оновлення зображення на екрані. Без цієї функції нам в результаті не відобразилось би зміщення, а одразу результат, де буде знаходитись трикутник. І остання функція циклу time.sleep(0.05) задає маленьку паузу.

Реакція об'єктів на події

Події - те, що відбувається під час роботи програми: рух миші, натискання на клавішу або робота з вікнами тощо.

Для обробки події необхідно створити функції, а потім повідомити модуль, про те, що її необхідно прив'язати до певної події, інакше кажучи, вона має викликатись, як тільки подія була виконана.

Нехай трикутник буде рухатись тільки після натискання на клавішу ENTER. Наша функція буде приймати єдиний аргумент (event), в який модуль буде передавати інформацію про подію. Тепер треба, щоб модуль викликав функцію при натискані клавіші. Для цього використовуємо функцію bind_all:

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_polygon(10,10,10,60,50,35)
def movetriangle(event):
canvas.move(1,5,0)
canvas.bind_all('<KeyPress-Return>', movetriangle)
t.mainloop()

Якщо ми хочемо змінювати напрям руху нашого прикутника в залежності від натискання кнопок, треба змінити код:

def movetriangle(event):
if event.keysym == 'Up':
canvas.move(1,0,-3)
if event.keysym == 'Down':
canvas.move(1,0,3)
if event.keysym == 'Left':
canvas.move(1,-3,0)
else:
canvas.move(1,3,0)

Об'єкт event, який передається в функцію, має певний набор властивостей, один з яких keysym, в якому зберігається значення відповідної натиснутої клавіші. Тепер об'єднаємо весь код:

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
canvas.create_polygon(10,10,10,60,50,35)
def movetriangle(event):
if event.keysym == 'Up':
canvas.move(1,0,-3)
elif event.keysym == 'Down':
canvas.move(1,0,3)
elif event.keysym == 'Left':
canvas.move(1,-3,0)
else:
canvas.move(1,3,0)
canvas.bind_all('<KeyPress-Up>', movetriangle)
canvas.bind_all('<KeyPress-Down>', movetriangle)
canvas.bind_all('<KeyPress-Left>', movetriangle)
canvas.bind_all('<KeyPress-Right>', movetriangle)
t.mainloop()

Для чого ще потрібні ідентифікатори?

Потрібні для визначення фігури на полотні. Після створення фігур, кожній задається свій ідентифікатор 1,2,3,...,100,... . Ці ідентифікатори ми використовуємо для інших функцій, таких як move. Але для того, щоб не гадати де який ідентифікатор, краще всього зробити так:

from tkinter import *
t = Tk()
canvas = Canvas(t,width=400,height=400)
canvas.pack()
a = canvas.create_rectangle(10,10,40,40)
move(a, 5, 1)
t.mainloop()

Також можна за допомогою ідентифікатора і функції можна змінювати значення фігур таких як колір і колір рамки:

canvas.itemconfig(mytriangle, fill='blue')

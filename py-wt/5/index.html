#1 Розглянути теорію

Як створити рядок

Рядки завжди створюються одним з трьох способів. Ви можете використовувати одинарні, подвійні або потрійні дужки. 

  1. my_string = "Ласкаво просимо в Python!"
  2. another_string = 'Я новий текст тут...'
  3.  
  4. a_long_string = '''А це
  5. рядок, що займає
  6. декілька рядків'''
  7.  

Базові операції з рядками

Додавання рядків

За допомогою оператор + можна об'єднювати кілька рядків в один. Ця операція називається конкатенація. 

s1 = "Hello"
s2 = " world"
print(s1+s2) #Результат: 'Hello world'

При необхідності об'єднання рядки з числом його попередньо потрібно привести теж до рядка, використовуючи функцію str(): 

name = "John"
age = 30
"Name: " + name + ", age: " + str(age)#Результат: 'Name: John, age: 30'

Доступ по індексу

Рядок str - складається з набору символів. Тому можна отримати символ по індексу. Індекси починаються з 0.
riadok[1] #повертає другий символ рядка

В Python можна використовувати від'ємні індекси.
riadok[-3] #повертає третій з кінця символ рядка

Зрізи

Можна отримати декілька символів, частину рядка.
riadok[start:end]

  • riadok[1:4] - 1 - з якого символа починається зріз, 4 - на якому закінчується.
  • riadok[:4] - зріз з початку рядка.
  • riadok[1:] - зріз до кінця рядка.

Вивід інформації методом форматування

Оператор % 

            name = "Alex"
            'Hello, %s' % name #Результат: 'Hello, Alex'

f-рядки (Python 3.6+)

В Python версії 3.6 з'явився новий метод форматування рядків - "f-рядки", з його допомогою можна використовувати вбудовані вирази всередині рядків.
Він виглядє наступним чином:
print(f'Some information about - {name_zminna}')
Наприклад:
name = 'Volodymyr'
age = 20
print(f'My name is {name}. I am {age} y.o.')

Функції і методи для роботи з рядками

  • len(riadok) - довжина рядка (повертає кількість символів)
  • str(riadok) - перетворення числового або іншого типу до рядка
  • riadok.find('a', start, end) - шукаємо символ 'a' в рядку. Повертає індекс першого входження 'a' або -1 при відсутності.Пошук йде в межах від start до end (можна не задавати)
  • riadok.split(x) - розбиває рядок на підрядки за допомогою обраного роздільника x (наприклад, пробілу або коми). Повертає список підрядків (окремих слів).
  • riadok.join(x) - об'єднює рядок і підрядки за допомогою обраного роздільника x.
  • riadok.strip() - прибирає пробіли з обох боків рядка.
  • riadok.lstrip() чи riadok.rstrip() - прибирає пробіли тільки зліва чи справа.
  • riadok.isdigit() - рядок складається з чисел
  • riadok.upper() - зміна рядку до верхнього регістру (великими літерами)
  • riadok.lower() - зміна рядку до нижнього регістру (малими літерами)
  • riadok.capitalize() - зміна першої літери у верхній регістр, інших - в нижній.
  • riadok.replace(a, b) - заміна символів а на символи b.

Більше методів з прикладами коду можна знайти за посиланням.


Цикл for

Цикл for виконує ті ж самі задачі, що й цикл while. Головна відмінність між ними це те, що цикл for менше і займає набагато менше пам'яті. В подальшому при створенні великих програм необхідно звертати увагу на економічність програми.

Як будується цикл for

Цикл for будується наступним чином:
for i in range(0,10):
print('hello')
Вираз i in range(0,10) - значить, що ми створюємо змінну і, яка дійсна тільки в цьому циклі. І що вона проходить значення від 0 до 10-1, тобто 10 проходів.
Також цикл for можна використати наступним чином:
a = 'danylo'
for i in a:
print(i)

В цьому випадку ми проходимось по нашому рядку проходячи кожен символ і виводимо його на екран.
